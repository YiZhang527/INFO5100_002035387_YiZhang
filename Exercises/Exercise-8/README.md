# Exercise 8 – XML & JSON Parsing

## Overview
This Java program demonstrates parsing and manipulating XML and JSON data. It reads book data from files, prints all entries, adds a new book programmatically, and prints the updated results.

## Project Structure
- `books.xml` — Sample XML file with multiple `Book` entries
- `books.json` — Sample JSON file with similar structure
- `BookParser.java` — Main Java program to:
    - Parse XML using DOM parser
    - Parse JSON using `org.json` library
    - Print all books
    - Add a new book entry
    - Print updated data
